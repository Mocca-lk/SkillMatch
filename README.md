## Skill match JS
### Simulador de Compatibilidade com Vaga Front-End Júnior
<p><strong>Objetivo do Projeto: </strong>
</br>
Criar um sistema simples que compare as habilidades de uma pessoa candidata om os requisitos de algumas vagas fictícias de front-end júnior.
</p>
A aplicação analisará:
<ul>
    <li>Haiblidades do candidato</li>
    <li>Habilidades solicitadas pela vaga</li>
    <li>Habilidades pendentes</li>
    <li>Percentual de compatibilidade do candidato com a vaga</li>
    <li>Recomedações de estudo ao candidato</li>
</ul>
</p>
